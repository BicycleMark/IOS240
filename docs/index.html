Before you begin

This is an iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either the iOS Simulator or an iOS device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

Touch is a fundamental part of the iOS experience. Almost all user interaction with running applications are done by touching the screen, and iOS devices have very responsive displays that can handle 5 to 11 touches simultaneously. As developers, making good use of touch and gestures can have a huge impact on both ease of use of your applications as well as user enjoyment.

Objectives

  1. Respond to touch events
  2. Handle multiple touch events
  3. Implement gestures

Lecture

IOS240 - Touch and Gestures.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Respond to touch events
    Exercise
    In this individual exercise, you'll add touch event handling to an existing iOS application.
  2. Create a multi-touch Paint app
    Exercise
    Next, you'll respond to multiple touch events to complete a multi-touch paint application.
  3. Use a gesture recognizer
    Exercise
    In this exercise you'll use a gesture recognizer to move an image within a view controller.
  4. Use multiple gestures simultaneously
    Exercise
    In the final exercise, you'll use three gestures simultaneously to transform an image on screen.
